# Issue #20: 図鑑画面の実装

### 背景 / 目的
相棒キャラクターをカラー表示し、その他8体をシルエット表示する図鑑画面を実装し、ユーザーに収集要素を提供する。

- **依存**: #8
- **ラベル**: `frontend`, `feature`

---

### スコープ / 作業項目

#### 1. 図鑑画面のUI作成
- `app/(tabs)/collection.tsx` を作成
- 3列グリッドレイアウトで9体のキャラクターを表示

#### 2. キャラクター表示ロジック
- 相棒キャラクター: カラー表示＋「相棒」バッジ＋レベル表示
- 未取得キャラクター: シルエット表示＋「???」＋「未取得」表示

#### 3. 収集率の表示
- 上部に収集率「1 / 9体」を表示
- Phase 1では相棒キャラクターのみ取得可能なため、常に「1 / 9体」

#### 4. キャラクター詳細表示（Phase 2以降）
- キャラクターカードタップで詳細モーダル表示
- 名前、説明、進化系統を表示

#### 5. シルエット画像の作成
- 各キャラクター画像のシルエット版を作成
- ImageコンポーネントのtintColorで黒く表示

---

### ゴール / 完了条件（Acceptance Criteria）

- [ ] `app/(tabs)/collection.tsx` で図鑑画面のUI作成完了
- [ ] 3列グリッドで9体のキャラクターを表示できる
- [ ] 相棒キャラクターはカラー表示＋「相棒」バッジ＋レベル表示
- [ ] 未取得キャラクターはシルエット＋「???」＋「未取得」表示
- [ ] 収集率「1 / 9体」を上部に表示できる
