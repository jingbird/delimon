# Issue #24: プライバシーポリシー・利用規約画面の実装

### 背景 / 目的
プライバシーポリシーと利用規約の表示画面を実装し、ユーザーが内容を確認できるようにする。

- **依存**: #23
- **ラベル**: `frontend`, `feature`

---

### スコープ / 作業項目

#### 1. プライバシーポリシー画面の作成
- `app/settings/privacy.tsx` を作成
- スクロール可能なテキストビューでプライバシーポリシー全文を表示

#### 2. 利用規約画面の作成
- `app/settings/terms.tsx` を作成
- スクロール可能なテキストビューで利用規約全文を表示

#### 3. プライバシーポリシーの内容作成
以下の項目を含む:
- 収集する情報（プロフィール情報、位置情報、配送履歴）
- 情報の利用目的（サービス提供、機能改善）
- 情報の公開範囲（ニックネームのみ公開、その他は非公開）
- 情報の保存期間
- 第三者への提供（原則として提供しない）
- お問い合わせ先

#### 4. 利用規約の内容作成
以下の項目を含む:
- サービス概要
- ユーザーの責任
- 禁止事項
- サービスの変更・終了
- 免責事項
- 準拠法・管轄裁判所

#### 5. ナビゲーション
- 戻るボタンで設定メニューへ戻る

---

### ゴール / 完了条件（Acceptance Criteria）

- [ ] `app/settings/privacy.tsx` でプライバシーポリシー画面を作成完了
- [ ] `app/settings/terms.tsx` で利用規約画面を作成完了
- [ ] スクロール可能なテキストで全文を表示できる
- [ ] 戻るボタンで設定メニューへ戻ることができる
- [ ] プライバシーポリシーにプロフィール情報の収集・利用・公開範囲を明記している

---

### テスト観点

#### 手動テスト
- 設定メニューから「プライバシーポリシー」をタップし、画面が表示されることを確認
- プライバシーポリシーの全文が表示され、スクロール可能であることを確認
- 戻るボタンで設定メニューへ戻ることを確認
- 設定メニューから「利用規約」をタップし、画面が表示されることを確認
- 利用規約の全文が表示され、スクロール可能であることを確認

#### 検証方法
1. 設定メニューから「プライバシーポリシー」をタップ
2. プライバシーポリシー画面が表示されることを確認
3. スクロールして全文を確認
4. 戻るボタンをタップし、設定メニューへ戻ることを確認
5. 同様に「利用規約」も確認
