# Issue #23: 設定メニュー画面の実装

### 背景 / 目的
各種設定へのアクセスポイントとなる設定メニュー画面を実装し、ユーザーがアプリの設定を変更できるようにする。

- **依存**: #15
- **ラベル**: `frontend`, `feature`

---

### スコープ / 作業項目

#### 1. 設定メニュー画面のUI作成
- `app/settings/index.tsx` を作成
- リスト形式で各種設定項目を配置

#### 2. ユーザー情報の表示
- 上部にユーザー情報カードを表示:
  - ニックネーム#識別番号（例: ハコブー太郎#1234）
  - メールアドレス
  - 登録日（例: 2025/11/06 登録）

#### 3. 設定項目の配置
以下の設定項目を配置:
- **プロフィール編集**: タップでプロフィール編集画面へ遷移
- **通知設定**: トグルスイッチでON/OFF切り替え
- **地図設定**: 地図の表示スタイル選択（標準、衛星写真）
- **データ収集設定**: トグルスイッチでON/OFF切り替え
- **プライバシーポリシー**: タップでプライバシーポリシー画面へ遷移
- **利用規約**: タップで利用規約画面へ遷移
- **アカウント削除**: タップでアカウント削除確認画面へ遷移
- **ログアウト**: タップでログアウト確認ダイアログ表示

#### 4. トグルスイッチの実装
- 通知設定、データ収集設定をトグルスイッチで実装
- 変更時にSupabaseのusersテーブルを更新

#### 5. ログアウト機能
- ログアウトボタンタップで「ログアウトしますか？」確認ダイアログ表示
- 「はい」でSupabase Authのサインアウト実行、ログイン画面へ遷移

---

- [ ] `app/settings/index.tsx` で設定メニュー画面のUI作成完了
- [ ] ユーザー情報（ニックネーム#識別番号、メールアドレス、登録日）を上部に表示できる
- [ ] プロフィール編集、通知設定、地図設定、データ収集設定のメニュー項目を配置できる
- [ ] プライバシーポリシー、利用規約、アカウント削除、ログアウトのメニュー項目を配置できる
- [ ] 各項目タップで対応する画面へ遷移、またはトグル切り替えできる

---

### テスト観点

#### 手動テスト
- 設定メニュー画面でユーザー情報が表示されることを確認
- プロフィール編集をタップし、プロフィール編集画面へ遷移することを確認
- 通知設定のトグルスイッチをON/OFF切り替え、Supabaseで更新されることを確認
- データ収集設定のトグルスイッチをON/OFF切り替え、Supabaseで更新されることを確認
- プライバシーポリシー、利用規約をタップし、対応する画面へ遷移することを確認
- ログアウトボタンをタップし、確認ダイアログが表示されることを確認
- 「はい」でログアウトし、ログイン画面へ遷移することを確認

#### 検証方法
1. ホーム画面右上の設定ボタンをタップ
2. 設定メニュー画面が表示されることを確認
3. ユーザー情報が正しく表示されることを確認
4. 各設定項目をタップ・切り替えし、正常に動作することを確認
5. ログアウトボタンをタップし、ログアウトできることを確認
